## How to Update GitHub Repository Description

1. Go to your repository on GitHub: https://github.com/NickScherbakov/english-vocabulary-t
2. Click on the ⚙️ (Settings icon) in the top-right corner of the About section
3. Paste your chosen description in the "Description" field
4. Optionally add topics: `vocabulary`, `english-learning`, `spaced-repetition`, `particle-animation`, `react`, `typescript`, `language-learning`, `sm2-algorithm`, `flashcards`, `russian-translation`
5. Click "Save changes"

The description will appear at the top of your repository and in GitHub search results.
